Welcome to the Magic Door Adventure!

You find yourself in a mysterious hallway with three magical doors. Each door leads to a unique adventure. Choose wisely!

How to Play

Select a door to begin your adventure. Each choice takes you to a new part of the story. To start over, refresh the page or click the "Restart" button at any time.
